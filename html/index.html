Cardano HTML DApp

Welcome to the Cardano HTML DApp example! This page showcases how to integrate a "Connect with Wallet" button into your HTML-based DApp using the cardano-connect-with-wallet-core library.

If you are looking for the core npm package or the React components, you can find them here:

Core npm Package React Components

To get started, just import the library:

                

                    <script src="https://cardano-foundation.github.io/cardano-connect-with-wallet/bundle-latest/index.js"></script>
GitHub Repository

Once you have the library set up, you can begin building your Cardano DApp by utilizing the provided functions and methods. Be sure to refer to the library's documentation for detailed usage examples and code snippets.

Remember to consider security best practices when handling user data and interacting with the blockchain. Always validate inputs, sanitize user input, and follow Cardano development guidelines to ensure the safety and integrity of your DApp.
